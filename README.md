# HA_paradise_transformer
Home Assistant control for the Northern International Paradise GL33210 Low Voltage Transformer

This transformer runs in AP mode, and has a fixed address of 192.168.4.1
You need to have the Home Assistant machine connect to the Paradise_XXXXXX wifi network first in order for it to work.

Installation
------------
Put the files in this repo under: /config/custom_components/paradise_transformer/
